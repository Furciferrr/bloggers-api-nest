## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.
[Vadim Setsko](https://twitter.com/vadimsetsko)

<p>You need to add next variables:</p>
<ul>
<li>MAIL_APP_PASSWORD</li>
<li>MONGO_URI</li>
<li>MONGO_URI_LOCAL</li>
<li>JWT_SECRET</li>
<li>JWT_EXPIRATION</li>
<li>JWT_REFRESH_EXPIRATION</li>
</ul>

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

```


## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov

```
